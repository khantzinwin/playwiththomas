AA နဲ့ DA ဘာကွာလဲ?


AA - ၀ယ်ယူတဲ့ game account ကို စက်ထဲထည့်ပြီး download လုပ်ရပါမယ်။ Game Down ပြီးလို့ game ဆော့တဲ့အခါ ကိုယ်ပိုင် playstation account နဲ့ဆော့လို့ရပါတယ်။ Wifi ချိတ်ပြီးဖြစ်ဖြစ် မချိတ်ဘဲနဲ့ဖြစ်ဖြစ် ဆော့လို့ရပါတယ်။


DA - ၀ယ်ယူသော game account ကို စက်ထဲထည့်ပြီး download လုပ်ရပါမယ်။ download လုပ်ပြီး အဲ့ Game ကို ဆော့တဲ့အခါ အဲ့ Game Accoun ထဲမှာဘဲဆော့ပေးရပါမယ်။ ကိုယ်ပိုင် Account ထဲမှာဆော့လို့မရပါ။ဆော့တဲ့အခါ wifi ချိတ်ပေးထားဖို့လိုအပ်ပေမယ့် လိုင်းကောင်းစရာလဲ မလိုသလို data လဲမကုန်ပါ။ background မှာ license key လေးကို refresh လုပ်ဖို့အတွက်ပါ။ wifi ပျက်သွားတဲ့အခါမျိုးဆိုရင်တောင် mobile data or cellular data နဲ့ hos လွှင့်ပြီးတောင် ဆော့ကစားနိုင်ပါတယ်။ Data မကုန်ပါ။
