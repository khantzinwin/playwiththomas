PS4 AA Game သွင်းနည်း

1. PS4 AA Installation

2. New User ကို နှိပ်ပြီး account ၀င်ရပါမယ်

3. အပေါ်က create a user နဲ့၀င်ရပါမယ်။ အောက်က guest mode နဲ့၀င်မိရင် ထည့်ထားတဲ့ account ပြောက်သွားပါလိမ့်မယ်။

4. Accept Button လေးကို နှိပ်ပေးလိုက်ပါ။

5. Next Button လေးကို နှိပ်ပေးလိုက်ပါ။

6. Sign In Manually နဲ့ ၀င်ပေးပါ။

7. mail ထဲမှာ ပေးထားတဲ့ sign in & password ကို ထည့်ပြီး sign in ၀င်ပေးရပါမယ်။

8. AA ၀ယ်ထားတာမို့လို့ ဒီအဆင့်ပေါ်လာခဲ့ရင် Change to This PS4 ကို နှိပ်ပေးလိုက်ပါ။

9. Account ထဲ၀င်သွားပြီဆိုရင် software icon ရဲ့ ညာဘက်အစွန်းကိုသွားလိုက်ပါ Library ကိုတွေ့ပါလိမ့်မယ် အဲ့ထဲကို ၀င်လိုက်ပါ။

10. Library ထဲ၀င်သွားရင်အောက်ဆုံးက purchaed ထဲကို ၀င်လိုက်ပါ။ ၀ယ်ထားတဲ့ game ကိုတွေ့ပါလိမ့်မယ်။ အကယ်လို့ Purhased ထဲက ညာဘက်အပေါ်ထောင့်လေးမှာ offline လို့ပေါ်နေရင် WiFi ကြောင့် game မပေါ်သေးတာပါ။ ခဏစောင့်ပေးလိုက်ပါ။ ပေါ်လာပါလိမ့်မယ်။

11. ၀ယ်ထားတဲ့ game ထဲကို၀င်ပြီး download button နှိပ်ပေးလိုက်ပါ။ game စဒေါင်းပါလိမ့်မယ်။

12. ps button နှိပ်ပြီး အပြင်ပြန်ထွက်ပါ။ Up Arrow လေးနှိပ်ပြီး ပုံထဲက power icon လေးထဲကို ၀င်ပါ။

13. မိမိ ကိုယ်ပိုင် PS Account ထဲကို ပြောင်းဖို့အတွက် switch user လေးကို နှိပ်ပေးပါ။

14. ဒီနေရာမှာ မိမိ ကိုယ်ပိုင် PS Account ထဲကို၀င်လိုက်ရင်ရပါပြီ။

15. မိမိကိုယ်ပိုင် PS Account ထဲရောက်ပြီဆိုရင် Up Arrow လေးနှိပ်ပြီး Notification ထဲကို ၀င်လိုက်ပါ။

16. download ထဲကို ဆက်၀င်ပေးပါ။

17. Game ကို ဘယ်လောက်ကြာကြာ ဘယ်နှ GB ဒေါင်းရမလဲကို အသေးစိတ်ကြည့်လို့ရပါပြီ။

18. Enjoy Your Game
