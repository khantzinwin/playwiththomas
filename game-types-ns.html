Primary Account နဲ့ Non-primary Account ဘယ်လိုကွာလဲ?


Primary Account

- မိမိကိုယ်ပိုင် user နဲ့ဆော့ကစားပေးရပါမယ်။ (ကျွန်တော်ပေးသော account user နဲ့ဆော့လို့မရပါ။)

- ကိုယ်ပိုင် user account နဲ့ Wifi ဖွင့်ပြီးဆော့လို့ရသလို ပိတ်ပြီးဆော့လို့လဲရပါတယ်။ (Online Multiplayer ပါဆော့လို့ရ)


Non Primary Account

- ပေးထားသော account user ထဲမှာဘဲဆော့လို့ရပါမယ်။ မိမိကိုယ်ပိုင် user နဲ့ဆော့လို့မရပါ။

- ဂိမ်းမဆော့ခင် wifi လေးခဏချိတ်ပေးရပါမယ်။ ဂိမ်းထဲဝင်လို့ nintendo logo လေးပေါ်ပြီဆိုတာနဲ့ ချက်ချင်း airplane mode on ပေးရပါမယ်။ (Wireless device များအတွက် setting ထဲက airplane ထဲမှာ bluetooth ကို on ပေးရပါမယ်။)


မှတ်ချက်

- Non-primary မှာ Airplane Mode On ရတယ်ဆိုတာ ဆော့နေရင်း user တိုက်ပြီး game ထဲကနေထွက်သွားမှာစိုးလို့ပါ။
- game ထဲပြန်ဝင်လိုက်ရင်တော့ ရပေမယ့် အဲ့လိုထွက်သွားရင် စိတ်အနှောက် အယှက် ဖြစ်မှာစိုးလို့ Airplane Mode On ပြီးဆော့ဖို့ကို recommend ပေးတာပါ။
- လုံးဝမဖြစ်မနေ on ရမယ်လို့ မဆိုလိုပါ။
- PWT က ရောင်းပေးတဲ့ account များသည် စျေးပေါတဲ့ crack account များ မဟုတ်တဲ့အတွက် ban ခံရနိုင်ချေ လုံးဝ chance မရှိပါ။ (FIFA ကို non-primary ဝယ်ပြီး transfer update မရမှာ စိတ်မပူပါနဲ့။ EA account ချိတ်ပြီး transfer update လုပ်လို့ရပါတယ်။)
- Primary ဘဲဖြစ်ဖြစ် Non-primary ဘဲဖြစ်ဖြစ် တစ်ခါဝယ်ယူပြီးတာနဲ့ ရက်အကန့်အသတ်မရှိဆော့လို့ရအောင် စီစဉ်ပေးထားပါတယ်။
- နှစ်ခုလုံးကတော့ ဈေးအခြေအနေပေါ်မူတည်ပြီး သူ့ဟာနဲ့သူဆော့ရတာအဆင်ပြေပါတယ်။
- အဓိကတော့ primary account ဝယ်ထားရင် ကိုယ်ပိုင် user account ထဲမှာဆော့ကစားပေးရမှာဖြစ်ပြီး Non-primary Account ဆိုရင် ပေးထားသော account ထဲမှာ ဂိမ်းစစဝင်ရင် wifi ချိတ် nintendo logo လေးပေါ်လာပြီဆိုတာနဲ့ airplane mode လေး on ပေးရုံပါဘဲ။
